
# PyTorch 基础学习项目

## 简介
这是我的第一个深度学习项目，用于熟悉 PyTorch 的 Tensor 操作和训练流程。

## 环境配置

### 1. 根据配置文件创建环境
conda env create -f environment.yml

### 2. 激活环境
conda activate pytorch_basics

## 项目结构

Proj_PyTorch_Basics/
    ├── environment.yml    # 环境配置文件
    ├── .gitignore         # Git 忽略规则
    └── README.md          # 项目说明书
