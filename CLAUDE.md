## Project Overview

A PyTorch deep learning basics project for learning core concepts through image classification. Phase 1 (Dataset → Transform → DataLoader → CNN → Training Loop → Data Augmentation) is complete. Currently entering Phase 2: transfer learning with pretrained models.

Uses modular design with reusable components in `src/` and learning demonstrations in `examples/`.

## Environment Setup

```bash
conda env create -f environment.yml
conda activate pytorch_basics
```

Key stack: Python 3.9, PyTorch 2.0.0, CUDA 11.8, torchvision 0.15.0, OpenCV 4.8, TensorBoard 2.13.

## Running Examples

All example scripts run standalone from the project root:
```bash
python examples/test.py
python examples/demo_classification.py
python examples/demo_dataloader.py
python examples/nn_conv2d.py
python examples/train_with_aug.py
python main.py
python train.py
```

TensorBoard logs (generated by some examples) can be viewed with:
```bash
tensorboard --logdir=runs
```

## Architecture

### `src/` - Reusable Modules

- **`src/data_modules/`** - Three PyTorch Dataset implementations:
  - `base.py` - `MyData`: PIL-based classification dataset, auto-maps folder names to integer labels
  - `classification.py` - `ClassificationDataset`: OpenCV-based variant returning numpy arrays
  - `detection.py` - `DetectionDataset`: YOLO-format detection dataset with separate image/label directories

- **`src/transforms/presets.py`** - Predefined augmentation pipelines (`train_transform`, `val_transform`) using ImageNet normalization stats, plus `load_image()` and `plot_compare()` utilities

- **`src/models/simple_cnn.py`** - `SimpleCNN`: 3×Conv+ReLU+Pool feature extractor + 2×Linear classifier, targets CIFAR-10 (3×32×32 → 10 classes). Shape chain: (B,3,32,32) → feature → (B,64,4,4) → flatten → (B,1024) → Linear → (B,10)

- **Lazy loading pattern**: `__init__.py` files use `__getattr__` for dynamic imports so OpenCV is not a hard dependency

### `examples/` - Learning Demonstrations

Progressive examples from basic tensor ops (`test.py`) through datasets, transforms, conv layers, nn.Module usage, to data augmentation experiments. Several examples write TensorBoard logs for visualization.

### Entry Points

- `main.py` - Loads hymenoptera dataset with DataLoader
- `train.py` - Complete CIFAR-10 training script using SimpleCNN with device management, 5-step training loop, eval/train mode switching, and model saving
- `examples/train_with_aug.py` - Augmentation experiment: baseline vs augmented transform comparison on CIFAR-10

### Datasets

- `hymenoptera_data/` - Ant/bee images in ImageFolder format (train/val splits)
- `data/` - Same dataset in YOLO annotation format
- `datasets/` - Downloaded public datasets (e.g., CIFAR-10), located at `../datasets` relative to project root
